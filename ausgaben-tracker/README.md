# Ausgaben-Tracker 💰

Einfaches Python-Programm zur Verwaltung persönlicher Ausgaben.

## Features
- Ausgaben mit Betrag und Kategorie hinzufügen
- Alle Ausgaben anzeigen mit Gesamtsumme
- **JSON-Speicherung** – Daten bleiben nach Programmende erhalten
- Fehlerbehandlung für ungültige Eingaben
- Komma und Punkt als Dezimaltrennzeichen
- Farbige Terminal-Ausgabe via colorama

## Technologien
- Python 3.14
- json (Datenspeicherung)
- os (Dateipfad-Prüfung)
- colorama (Terminal-Farben)

## Ausführen
pip install colorama
python kostenberechnung.py
