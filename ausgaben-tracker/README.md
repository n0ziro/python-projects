# Ausgaben-Tracker 💰

Einfaches Python-Programm zur Verwaltung persönlicher Ausgaben.

## Features
- Ausgaben mit Betrag und Kategorie hinzufügen
- Alle Ausgaben anzeigen mit Gesamtsumme
- **JSON-Speicherung** – Daten bleiben nach Programmende erhalten
- Fehlerbehandlung für ungültige Eingaben
- Komma und Punkt als Dezimaltrennzeichen
- Farbige Terminal-Ausgabe via colorama

## Technologien
- Python 3.14
- json (Datenspeicherung)
- os (Dateipfad-Prüfung)
- colorama (Terminal-Farben)

## Ausführen
pip install colorama
python kostenberechnung.py

## Gelernte Konzepte
- Funktionen (def save, def load)
- JSON lesen und schreiben
- try/except Fehlerbehandlung
- Listen und Schleifen
- if __name__ == "__main__"

## ⬇️ Download (Windows)

Du kannst das Programm direkt ausführen ohne Python zu installieren:

1. Lade die Datei [kostenberechnung.exe](kostenberechnung.exe) herunter
2. Doppelklick → Programm startet sofort
3. Keine Installation nötig

> **Hinweis:** Windows SmartScreen kann eine Warnung anzeigen.
> Klicke auf "Weitere Informationen" → "Trotzdem ausführen".
